This project will make the opera_install_file_lock.exe file found in the platforms/windows/selftests/data folder. If you compile this project in Release, it will replace the file there by default. 

The exe file created by this project is used to lock the platforms/windows/selftests/data/opera_installer_locked_file.odt file for during the selftest called platforms/windows/selftests/operainstaller.handleinfo.ot
