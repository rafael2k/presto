Opera 12,10 artıq buradadır!

Bu yeni Opera brauzer versiyası sizin internetdə gəzmə təcrübənizi daha cəld və asan etmək üçün tamamilə təzə təkmilləşmələr siyahısını təgdim edir! Vurğulamaların yalnız bəziləri buradadır:

Biz səhifələrin yüklənməsi üçün tələb olunan vaxtı qısaldan SPDY şəbəkə standartı üçün dəstək əlavə edirik. Gmail, Twitter.com və digər populyar saytlar artıq SPDY standartından istifadə edirlər.

Yeni Opera ilə həyatınızı daha da asanlaşdırmaq üçün istifadə edə biləcəyiniz daha çox genişlənmə vardır. addons.opera.com səhifəsində təsnifat və ya populyarlığa görə axtar.

Opera 12.10 daha çox əməliyyat sistemi və platformalarla daya yaxşı işləyir. Biz Mac istifadəçiləri Mountain Lion-un quraşdırılmış paylaşma funksiyası daxil olan Opera ilə OS X Mountain Lion-un yeni imkanlarından faydalandığı zaman Windows 8 Classic və Windows 7 əməliyyat sistemində Opera üçün əsas sensor dəstək daxil etmişik.

Daha çox bilmək istəyirsiniz? Bu versiyada olan yeniliklərə baxmaq üçün Yeniliklər səhifəsini açın və ya texniki yaxşılaşdırmaların tam siyahısını görmək üçün buraya klikləyin.
