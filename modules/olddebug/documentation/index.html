Olddebug module documentation

The 'olddebug' module is used for debugging parts of the URL module. Module owner is unaware of any other uses for this module.

If you are using functionality in the 'olddebug' module for anything else than debugging URL, please let the module-owner know as the plan for 'olddebug' is to provide the functionality required through debug, probetools and scope.

Knowing what this module is used for will help in this transition.

Known problems

The global Opera object is not always created when it should. This is decided to be of little importance in the long run. If you do have issues with this, please let it be known!

API

As this module is to be deprecated, you are not enchouraged to use it, and documentation for the API will not be provided. You will have to learn the hard way, sorry...
