Introduction

The XSLT module provides support for the W3C XSLT 1.0 specifications. The module depends heavily on the XPath implementation in the xpath module.

API documentation

For detailed information on how to use the XSLT module in Opera and the module's public API, please refer to the API documentation. The documentation needs to be generated by Doxygen.
