libcrypto - A pure crypto library

This module is a pure crypto library, and is independent of the libssl module. It may be used e.g. when the full libssl implementation causes a footprint problem.

Xml signing tutorial

Three Xml signing tutorials, with certificate, without certificate, and widget signing. An xml signature can be used to sign any file type.
