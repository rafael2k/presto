Introduction

The DOM module provides support for various W3C DOM specifications (mostly DOM Level 1 and 2, plus selected parts of DOM Level 3) and for legacy JavaScript interfaces.

Current information about the DOM module.

API documentation

For detailed information on how to use the DOM module in Opera and the module's public API, please refer to the API documentation. The documentation needs to be generated by Doxygen. Documentation for the latest official module release is available from the intranet.
