Tutorial 17 - Delays

// tutorial17.ot
group "tutorial_17";

test("Delay 5 seconds after running this test") delay 5;
{
  verify(1 == 1);
}

test("No delay")
{
  verify(3 == 3);
}

test("Delay 3 seconds before running this test") delay pre 3;
{
  verify(2 == 2);
}

Delay can be used to delay before or after (default) running a test. The delay is given in seconds. Remember that using delays will make running the tests taking longer time, so you should only use delay if you have a very good reason.
