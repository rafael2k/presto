Scheduler regression tests

Inline script tests

These tests test the correct scheduling of inline scripts, especially in relation to processing data written using document.write.

  • Inline scripts writing to the same document.
  • Inline scripts writing to another document (in an iframe).

Timeout script tests

These tests test so that timeout and interval threads are executed at the right time, and are properly stopped by clearTimeout/clearInterval.

  • Tests setTimeout and clearTimeout.
  • Tests setInterval and clearInterval.

Bugs?
